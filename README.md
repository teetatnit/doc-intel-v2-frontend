## Usage

### Run By cmd / PowerShell / Bash

```bash
$ git clone https://chemdevgl.southeastasia.cloudapp.azure.com/DI/di-frontend.git
$ cd di-frontend
$ npm install
$ npm start         # visit http://localhost:8000
```
